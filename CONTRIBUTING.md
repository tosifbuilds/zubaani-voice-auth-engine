# 🤝 Contributing to Zubaani

First off, thank you for considering contributing to Zubaani! 

The open-source community thrives on collaboration. Whether you are fixing a typo, optimizing Firebase latency, or redesigning the UI for better accessibility—your contribution matters.

## ⭐ The Easiest Way to Contribute
If you believe in making the internet accessible for everyone, the simplest way to support this mission is to **give this repository a Star ⭐**. 
It takes one click, but it helps this project reach thousands of other developers who can build upon this vision.

## 🛠️ How Can I Contribute?

### 1. Reporting Bugs
Found a glitch in the voice matrix or a loop in the auth flow? 
* Check the existing issues to ensure it hasn't been reported.
* Open a new Issue, use a clear title, and describe the exact steps to reproduce the bug.

### 2. Suggesting Enhancements
Have an idea to make the AI voice sound more natural? 
* Open an Issue and tag it as an `enhancement`. 
* Explain **why** this feature would improve the digital accessibility ecosystem.

### 3. Pull Requests (Your Code Here)
1. Fork the repo and create your branch from `main`.
2. Write clean, modular code.
3. Update the documentation if you are changing core architecture.
4. Issue that pull request!

## 🧠 Code of Conduct
By participating in this project, you agree to abide by our Code of Conduct. We are building a safe, inclusive ecosystem for all developers.
